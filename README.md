# miniworld_virus_all

#### 介绍
防止盗版游戏《迷你世界》运行

#### 安装教程
目前自动安装器因BUG无法完成自动安装，请手动安装：

1.在\lib\文件夹选择需要的版本(1.0自动关机，2.0崩溃，3.0打不开迷你，3.1在3.0的基础上增加了热备份，有两个文件)

2.复制到C:\Users\（用户名）\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup中

#### 使用说明
软件开机自启，没了（手动滑稽）
